# Cute Asp Framework (ver.3.9.2)

您看到的是一个小巧、灵活、高效的为快速开发而整理设计的Asp开发框架。

## 使用说明
    
框架内的方法和属性全部是已Tpub.为前缀，需要用到的类库直接在文件include引用即可，不需要再实例化。

示例:
```asp
echo Tpub.Md5.Encode("admin")
```
